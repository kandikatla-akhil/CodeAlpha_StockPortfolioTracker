# Stock Portfolio Tracker 📈

A simple Python-based Stock Portfolio Tracker that allows users to calculate their total investment value using manually defined stock prices.

---

## 🚀 Features
- Add stock symbols and quantities
- Uses hardcoded stock prices
- Calculates total investment value
- Displays portfolio summary
- Saves portfolio details into a text file
- Beginner-friendly project

---

## 🛠️ Technologies Used
- Python
- Dictionaries
- Loops
- Conditional Statements
- File Handling

## To Run Use This Command
- python3 stock_tracker.py
